# 交付目标：

根据阳老师周五的反馈，无论大家写的怎么样，他都会改的，所以我们的交付物在尽量保证原本意思的基础上做到以下几点即可，尽量多，不要少。
- 互相校对文字内容,避免错别字
- 对自己不理解的术语增补解释；（可以使用 site:www.yangzhiping)
- 对阳老师提及的一些过于抽象的理论或者黑话（术语）增补一个通用的案例；
- 用写匠检查错别字

另外，第二轮的交付物要和第一轮的独立开，最终阳老师觉得哪个版本他改起来更顺手就选择哪个版本的。
